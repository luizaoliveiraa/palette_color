Projeto criado com o objetivo de:
 - Ver uma paleta de 5 cores (aleatórias) quando a página carrega.
 - Incluir um botão que gere uma nova paleta.
 - Ver os códigos hexadecimais de cada cor.
 - Copiar o código hexadecimal de cada cor.
 - Bloquear uma cor, ao gerar novas paletas, as cores bloqueadas devem permanecer.
